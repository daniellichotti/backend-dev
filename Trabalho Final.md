# Trabalho

Desenvolva uma API usando Fastify (Assunto da sua escolha) com todas boas práticas aprendidas. É necessário documentação via Swagger. 
A API deve conter todos os métodos HTTP aprendidos (CRUD completo) e deve estar conectada a um banco de dados MySQL.
Os elementos do banco devem ter pelo menos 4 atributos (Deve conter inteiro e texto e o id deve ser gerado via UUID como feito em aula.).
